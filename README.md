# Menedżer Haseł

1. aktywuj venva
2. zainstaluj wymagania: `pip install -r requirements.txt`
3. zrob migracje
   ```bash
   python manage.py makemigrations
   python manage.py migrate
   ```
4. w przegladarce wejdz w rozszerzenia
5. wlacz tryb dewelopera w prawym rogu
6. kliknij 'zaladuj rozpakowane' i dodaj tam folder wtyczka/ z projektu
